📦 SIRIUS Ui 百科

适用于 Minecraft 26.1.1 | 数据驱动UI框架（开发中）

SIRIUS Ui 是一个通过 JSON 配置和纹理资源自定义 Minecraft 标题屏幕与游戏选择屏幕的模组。无需编写 Java 代码，即可实现图片、按钮、背景、动态响应式布局。

📌 当前版本支持两个屏幕：标题屏幕 (title_screen) 和 游戏选择屏幕 (game_selection_screen)。模组名称为 SIRIUS Ui。

概述

SIRIUS Ui 将原版 Minecraft 标题屏幕和游戏选择屏幕的硬编码界面替换为 JSON 驱动的动态渲染系统。资源包可覆盖以下文件：

  • assets/sirius_ui/screens/title_screen.json — 标题屏幕布局
  • assets/sirius_ui/screens/game_selection_screen.json — 游戏选择屏幕布局
  • assets/sirius_ui/buttons/*.json — 按钮样式定义
  • 任意路径的 PNG 纹理图片

核心特性：

  • 图片展示（logo 元素）
  • 交互按钮（normal / hovered / disabled 三态纹理）
  • 动态表达式坐标（基于屏幕宽高及元素自身尺寸）
  • 按钮动作：打开界面、退出游戏、返回上一级
  • 背景全景图控制（原版全景、纯黑、自定义图片）
  • 自定义背景图层 graphicsDraw（彩色矩形，支持表达式）
  • 静态文本元素 text

资源包结构

资源包文件夹层级必须遵循以下约定：

📦 你的资源包/
└── 📁 assets/
    └── 📁 sirius_ui/
        ├── 📁 screens/
        │   ├── 📄 title_screen.json
        │   └── 📄 game_selection_screen.json
        ├── 📁 buttons/
        │   ├── singleplayer.json
        │   ├── multiplayer.json
        │   └── ...
        └── 📁 textures/
            └── 📁 gui/
                ├── custom_logo.png
                ├── btn_normal.png
                └── ...
└── 📄 pack.mcmeta

pack.mcmeta 示例：

{
  "pack": {
    "pack_format": 26,
    "description": "SIRIUS Ui 自定义界面包"
  }
}

屏幕配置

每个屏幕 JSON 的根对象为 screen，其下可设置：

  • panorama — 背景配置
  • graphicsDraw — 背景图层数组
  • elements — 界面元素数组
  • copyright — 版权文字（仅标题屏幕有效）

背景 (panorama)

支持三种形式：

  • "panorama": "vanilla" — 原版动态全景图
  • "panorama": "none" — 纯黑背景
  • "panorama": { "type": "image", "texture": "sirius_ui:textures/gui/background.png" } — 自定义图片（按比例填充居中）

元素 (elements)

elements 数组包含以下类型的对象：

  • logo：显示静态图片。必须字段：texture, position, size。
  • button：交互按钮。必须字段：style, position, size, action。可选：enabled（默认 true）、text_key（覆盖样式文本）。
  • text：静态文本。必须字段：text_key, position。可选：color（默认 0xFFFFFF）、shadow（默认 true）。

背景图层 (graphicsDraw)

graphicsDraw 是一个数组，每个元素定义一个彩色矩形，绘制在背景与普通元素之间。每个图层字段（均支持表达式）：

  • x, y — 矩形左上角坐标
  • width, height — 矩形宽高
  • color — 颜色字符串，支持 "#RRGGBB"、"0xRRGGBB"、"0xAARRGGBB"

示例：底部红色装饰线

{
  "x": "0",
  "y": "height - 4",
  "width": "width",
  "height": "4",
  "color": "#AA0000"
}

按钮样式 (button style)

按钮样式文件（如 singleplayer.json）包含顶层 button 对象：

{
  "button": {
    "text_key": "menu.singleplayer",
    "texture": {
      "normal": "minecraft:widget/button",
      "hovered": "minecraft:widget/button_highlighted",
      "disabled": "minecraft:widget/button_disabled"
    },
    "text_color": 16777215,
    "hover_text_color": 16777088,
    "disabled_text_color": 10526880,
    "text_offset_x": "center",
    "text_offset_y": "(height - elementHeight)/2"
  }
}
  • text_key — 本地化键名，显示在按钮中央。
  • texture — 三种状态的纹理标识符（不含 .png 扩展名，纹理位于 assets/namespace/textures/gui/sprites/）。
  • text_color / hover_text_color / disabled_text_color — ARGB 颜色（十进制整数）。
  • text_offset_x / text_offset_y — 文本相对于按钮左上角的偏移（支持表达式）。也可使用 hovered_text_offset_* 和 disabled_text_offset_* 分别指定状态偏移。

动作 (action)

按钮的 action 字段支持以下类型：

  • 打开界面：{ "type": "open_screen", "screenId": "singleplayer" }
    预置 screenId：
    • singleplayer — 单人游戏（世界选择）
    • multiplayer — 多人游戏
    • options — 选项
    • realms — Realms 主界面
    • selectGame — 游戏选择界面（本模组自定义）
    • language — 语言设置
    • accessibility — 辅助功能
  • 退出游戏：{ "type": "quit_game" }
  • 关闭当前界面（返回）：{ "type": "close_screen" }

坐标系统与表达式

坐标系原点

原点 (0,0) 位于屏幕左上角，X 轴向右为正，Y 轴向下为正。

▲ 原点位于左上角，X 向右，Y 向下。

元素锚点

position 字段定义元素左上角相对于屏幕的坐标，而非中心。

⚠️ 常见误解："x": "width/2", "y": "height/2" 不会使元素居中，而是将左上角置于屏幕中心。

正确居中的表达式：

"x": "(width - elementWidth)/2"
"y": "(height - elementHeight)/2"

表达式变量与运算

支持以下变量：

  • width — 屏幕宽度（缩放后）
  • height — 屏幕高度
  • elementWidth — 元素宽度（来自 size.width）
  • elementHeight — 元素高度（来自 size.height）

支持运算符 + - * / ( )，除法为整数除法。快捷语法：

  • "center" → 等价于 "(width - elementWidth)/2"（仅对元素的 x 有效）
  • "bottom-20" → 等价于 "height - elementHeight - 20"（仅对元素的 y 有效）

不同尺寸元素对齐技巧

  • 左/右/水平居中：所有元素使用相同 x 表达式即可。
  • 底部对齐："y": "baseY - elementHeight"，baseY 为目标基线。
  • 纵向堆叠：手动累加 y 值，如 "y": "100 + 20 + 10"（按钮1高20，间距10）。
核心公式：左上角坐标 = 目标参考点 - 元素尺寸偏移（如右对齐：参考点 = width，偏移 = elementWidth）。

配置示例

以下是一个完整的标题屏幕配置（自定义Logo、三个按钮、顶部标题）：

{
  "screen": {
    "panorama": { "type": "image", "texture": "sirius_ui:textures/gui/background.png" },
    "graphicsDraw": [
      { "x": "0", "y": "0", "width": "width", "height": "height", "color": "#0A0A0A" },
      { "x": "0", "y": "height - 4", "width": "width", "height": "4", "color": "#FFAA00" }
    ],
    "elements": [
      {
        "type": "text",
        "text_key": "auui.title",
        "position": { "x": "center", "y": 12 },
        "color": 16777215,
        "shadow": true
      },
      {
        "type": "logo",
        "texture": "sirius_ui:textures/gui/fancy_logo.png",
        "position": { "x": "center", "y": 40 },
        "size": { "width": 300, "height": 90 }
      },
      {
        "type": "button",
        "text_key": "menu.singleplayer",
        "style": "sirius_ui:buttons/custom_play.json",
        "position": { "x": "center", "y": "(height - elementHeight)/2 - 40" },
        "size": { "width": 200, "height": 24 },
        "action": { "type": "open_screen", "screenId": "singleplayer" }
      },
      {
        "type": "button",
        "text_key": "menu.multiplayer",
        "style": "sirius_ui:buttons/custom_multi.json",
        "position": { "x": "center", "y": "(height - elementHeight)/2" },
        "size": { "width": 200, "height": 24 },
        "action": { "type": "open_screen", "screenId": "multiplayer" }
      },
      {
        "type": "button",
        "text_key": "menu.quit",
        "style": "sirius_ui:buttons/custom_quit.json",
        "position": { "x": "center", "y": "(height - elementHeight)/2 + 40" },
        "size": { "width": 200, "height": 24 },
        "action": { "type": "quit_game" }
      }
    ]
  }
}

故障排除

问题现象可能原因与解决方法
检查 action 和 screenId；确认按钮 enabled 为 true。
路径错误或文件缺失；检查命名空间和 .png 扩展名。
仅允许 width, height, elementWidth, elementHeight；检查拼写。
检查 graphicsDraw 中的颜色格式（如 #RRGGBB）和坐标范围。
确认 panorama.type 为 "image"，texture 路径包含 .png 扩展名，且文件存在。
🔧 按 F3+T 可重载资源包，无需重启游戏即可刷新 UI 布局。
